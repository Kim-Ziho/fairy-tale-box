## JavaScript

javascript (ECMAScript)
Script Language
- 프로그램이 실행되는 런타임에 실행된다

- 자바스크립트의 사용 범위는 굉장히 넓다!

## javaScript의 문법
javascript의 자료형
자료형 (Data Type)
- 변수의 데이터가 대입되는 시점에 해당변수의 자료형이 결정됨(Dynamic Typing 동적 타이핑)
1. number type
2. string type
3. boolean type
4. null type
5. undefined type
6. array type
    1. number 타입으로민 이루어진 배열
    2. string 타입으로만 이루어진 배열
    3. number 타입과 string 타입을 함께 사용한 배열
    4. 다양한 자료형들과 함께 사용한 배열
7. object type
    1. 값으로 string 타입만을 사용한 객체
    2. 값으로 number 타입만을 사용한 객체
    3. 값으로 다양한 자료형들을 함께 사용한 객체
    4. 값으로 객체를 사용한 객체
