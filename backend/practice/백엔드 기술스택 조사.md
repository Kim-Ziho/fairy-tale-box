# 상현

## 본체(라즈베리파이 or 젯슨나노)

web에서 출력되는 데이터를 본체에 저장해 놓는다!(ex. background img, main character, etc.)

⇒ 동화 데이터를 서버에서 다운받아 올바른 경로에 저장해야 함!

본체가 실행되면 자동으로 ‘프로필 등록 & 선택’ 페이지로 이동하게!

## AWS

front-end : react로 서버 실행 & 관리자 사이트는 인터넷으로 접속 가능하게!
